## document extraction pipline using docling 
This is a document extraction pipeline using docling. It is designed to extract information from documents and store it in a structured format. The pipeline consists of several steps, including document parsing, information extraction, and data storage.
it is for educational purposes
### Installation
I am using python 3.10 for this project. You can create a virtual environment and activate it before installing the dependencies, OR you can use anaconda to manage your environment and dependencies.
## anaconda
To create a virtual environment using anaconda, run the following command:
```bash
conda create -n document-pipeline python=3.10
```
To activate the virtual environment, run the following command:
```bash
conda activate document-pipeline
```
to deactivate the virtual environment, run the following command:
```bash
conda deactivate
``` 
To install the required dependencies, run the following command:
```bash
pip install -r requirements.txt
```
